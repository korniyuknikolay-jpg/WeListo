# 🛒 WeListo — Совместный список покупок

Мобильное приложение на **React Native (Expo)** для совместного управления списками покупок в реальном времени.

## ✨ Основные функции

- 🔐 **Авторизация**: Регистрация и вход через Firebase Auth.
- ⚡ **Real-time синхронизация**: Мгновенное обновление списка у всех участников (Firestore).
- 🤝 **Совместный доступ**: Присоединение к спискам по **короткому invite-коду** (6 символов).
- ✏️ **Редактирование товаров**: Долгое нажатие для редактирования названия и заметки.
- ✍️ **Индикатор печати**: Видно, когда другой участник добавляет товар.
- 🗑️ **Подтверждение удаления**: Защита от случайного удаления списков.
- 🌓 **Темная тема**: Поддержка темного режима (Zustand + Persist).
- 🌍 **Локализация**: Интерфейс на русском и английском языках (i18next).
- 📋 **Копирование кода**: Быстрое копирование invite-кода в буфер обмена.

## 🛠 Технологический стек

- **Frontend**: React Native (Expo) 54, React 19
- **State Management**: Zustand 5 (с сохранением в AsyncStorage)
- **Backend/Database**: Firebase (Firestore, Auth, Storage)
- **Navigation**: React Navigation 7 (Stack Navigator)
- **i18n**: i18next 26 + react-i18next 17
- **Clipboard**: expo-clipboard

## 🚀 Как запустить проект

### Требования

- Node.js (рекомендуется LTS)
- Expo CLI (`npm install -g expo-cli`)
- Android Studio / iOS Simulator или физическое устройство с Expo Go

### Установка и запуск

1. Склонируйте репозиторий.
2. Установите зависимости: `npm install`.
3. Создайте файл `.env` в корне и добавьте свои ключи Firebase (см. `.env.example`).
4. Запустите проект: `npx expo start -c`.

### Установка и запуск проекта с тунелем

1. Скачать библиотеку в проект: `npm i -g @expo/ngrok`.
2. Запустите проект: `npx expo start --tunnel -c`.
